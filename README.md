# 💰 Expense Tracker App

A fully functional expense tracker web application built using JavaScript with local storage and analytics features.

## 🔥 Features
- Add, edit, and delete expenses
- Category-based filtering
- Total expense calculation
- Category breakdown
- Monthly summary analytics
- Data stored using local storage (persistent)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (DOM + Local Storage)
